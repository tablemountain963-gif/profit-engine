# Explanation Problem Prompt Pack
## How to Use
This prompt pack is designed to help you work with LLMs (Claude, GPT, Gemini) in the Explanation Problem domain. Each prompt is labeled by use case (e.g., research, drafting, analysis, automation) and is copy/paste ready. To use this pack, simply select a prompt that fits your needs and paste it into your LLM interface. You can then use the response to inform your work, whether that's researching a topic, drafting a document, analyzing data, or automating a task. Be sure to review and refine the prompts as needed to get the most out of your LLM.
